#  JavaScript Quiz-Score

##  Overview
This is an interactive quiz application built using **HTML, CSS, and JavaScript**.

The app presents questions one at a time, allows users to select answers, and provides immediate feedback. At the end, the final score is displayed with an option to restart the quiz.

---

##  Features
- Start Quiz screen
- Multiple-choice questions
- Instant feedback (correct/incorrect)
- Timer for each question
- Progress bar
- Score tracking
- Restart functionality

---

##  Technologies Used
- HTML (structure)
- CSS (styling)
- JavaScript (logic & interactivity)

---

##  Project Structure
Quiz-Score/
│── index.html
│── style.css
│── script.js
│── README.md


##  How It Works
- Questions are stored in an array of objects
- JavaScript dynamically creates answer buttons using `createElement()`
- Event listeners detect user selections
- Score updates when correct answers are selected
- Timer uses `setInterval()` for countdown
- Progress bar updates as user moves through questions

---

## ▶ How to Run
1. Open the project folder
2. Open `index.html` in your browser
3. Click **Start Quiz**
